====How to set up express in a directory====	
express -c stylus
npm install -d

===How to set up Heroku ===
-Create a profile of the format "web: node [name_of_js_file_in root directory_]"
-test application locally "Foreman start"
-Store app in Git:
	- git init
	- git add .
	- git commit -m "init"

-Heroku create
-Git push heroku master
